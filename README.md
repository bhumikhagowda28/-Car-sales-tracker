# -Car-sales-tracker
"Python app for car sales check "
This Python program is a menu-driven application designed to help track and analyze car sales data. It uses a dictionary to store sales figures for different car models and provides three main functionalities:
1.View Total Sales
- Calculates the sum of all car sales stored in the dictionary.
- Useful for quickly checking overall performance.

2.View Sales by Car Model
- Allows the user to input a specific car model name.
- Displays the number of units sold for that model.
- If the model doesn’t exist in the dataset, it returns a “Model not found” message.

3.View Best-Selling Car- Identifies the car model with the highest sales.
- Helps determine which car is performing best in the market.

4.Exit Option
- Ends the program when the user chooses to quit.
